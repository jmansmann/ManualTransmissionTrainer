gearHeader.arff is used to allow the TransmissionTrainer to create new matrix of incoming data from car.
gear_train_data.arff is the file used to train the Decision Tree.

In Test Arff Files:
gear_test_long.arff is the test file created from an OBDWiz file that was not able to be used to test a specific gear - has multiple gears in it.
gear_test_first.arff and similarly named files are the files used to test the accuracy of the tree for each gear.
